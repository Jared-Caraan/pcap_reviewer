## Try-except with else

Create a new [Jupyter Notebook](https://jupyter.org/try-jupyter/tree/). and call it _exceptions-else_.

So the `except` branch was not executed, but instead we could see the `else` branch in action.

The rule is that the `else` block will be executed if the `except` block doesn't get executed. In other words, the `else` block is run when no exception is raised in the try-block and that's pretty much it.
